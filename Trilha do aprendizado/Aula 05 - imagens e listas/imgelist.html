Imagens e Listas

Imagem

Imagem representação visual de uma pessoa, objeto, paisagem. Imagem digital é a representação de uma imagem bidimensional usando números binários codificados de modo a permitir seu armazenamento, transferência, impressão ou reprodução, e seu processamento por meios eletrônicos.

Existem formatos de imagens que são padrões de codificação, compressão e definição que determinam o tipo de arquivo gráfico, tais como,

  • .BMP
  • .TIFF
  • .GIF
  • .SVG
  • .ICO
  • .JPEG
  • .PNG

As licenças Creative Commons são várias licenças públicas que permitem a distribuição gratuita de uma obra protegida por direitos autorais.

Para a realização da marcação de imagens usamos a tag <img> e são seus parametros q definem aas imagens e suas propriedades um exemplo de parametro o mais importante é o SRC="" q referencia a imagem, além deste temos width="" q define a largura, height="" q define a altura e alt q descreve a imagem.

Além de imagens temos favicons que são os icones que aparece na barra de titulo, são escritas no head, para a marcação usamos <link> e em seus parametros temos de exemplo o mais importante o href="" que diz ao navergador o caminho para o icone, alem disso temos type="" que define o tipo de imagem.

Marcação de listas

As tags de listas se dividem em: listas não-ordenadas, listas ordenadas e lista detalhada.

Listas não-ordenadas

Para marcação usamos a tag <ul> e então <li>, para cada item listado deve haver <li> e </li>, apó listar todos fechamos com </ul>

Listas ordenadas

Para marcação usamos a tag <ol> e então <li>, para cada item listado deve haver <li> e </li>, apó listar todos fechamos com </ol>. A tag <ol> possui o parametro type="" que define como será listado

Listas detalhadas

Para marcação usamos a tag <dl> e então <dt>, para cada item listado deve haver <dt> e </dt>, e para os detalhamentos usamos <dd> e </dd>, após listar todos fechamos com </dl>
