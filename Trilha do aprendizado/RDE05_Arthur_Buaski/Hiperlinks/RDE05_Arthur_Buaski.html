Hipertexto em HTML

É uma referência dentro de um documento em hipertexto para outras partes do mesmo documento, outros documentos, arquivos ou endereço eletrônico Os links podem ser remotos, internos, locais, com endereço de correio ou arquivos

A marcação de hiperlinks acontece utilizando a tag <a> e a propriedade href que faz a referência para ao conteúdo desejado.

Em html temos varias maneiras de por hiperlinks:

  1. Hiperlink para um site externo
  2. Hiperlink para um site interno
  3. Hiperlink dentro do próprio documento
  4. Hiperlink em imagem
  5. Hiperlink para e-mail

Hiperlink externo
