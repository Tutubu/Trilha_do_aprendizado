Table:

Definição de uso:

A tag <table> é utilizada para a criação de tabelas em html, junto com ela usamos os comandos:

  • <tr> - Define as linhas;
  • <th> - Define o cabeçalho;
  • <td> - Define os valores.

É importante que o número de <td> em cada <tr>, seja igual ao número de <th>

Opcional: caso deseje colocar borda adicione o parametro border="" na tag <table> dentro das aspas você coloca o numero que deseja para ser a grossura da borda

Exemplos:

Horas Minutos Segundos
1 60 3600
2 120 7200

Address:

Definição de uso:

A tag <address> define as informações de contato do autor/proprietario de um documento ou artigo. O texto geralmente será posto em itálico e os navegadores sempre adicionam uma quebra de linha antes e depois dos elementos de <address>

Exemplos:

Bdo:

Definição de uso:

A tag <bdo> é usada para substituir a direção do texto. BDO significa Bi-Directional Override (Substituição Bi-Directional). Dentro da tag ainda usamos o parametro dir="" para mostrar a direção que o texto deve ir

  • rtl - right to left(direita para a esquerda)
  • ltr - left to right(esquerda para a direita)

Exemplos:

Este texto esá de direita para a esquerda

Retornar

Pre

Definição de uso:

A tag <pre> define um texto pré-formado. Seus elementos são exibidos em uma fonte de largura fixa e o texto preserva espaços e quebras de linha. O texto será exibido exatamente como escrito no código-fonte HTML.
